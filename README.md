# 🐺 FURIA UTFPR - Hours Tracker

Sistema de controle de horas desenvolvido para gerenciamento de atividades e registro de horas trabalhadas.

---

## 📌 Sobre o projeto

O **FURIA UTFPR - Hours Tracker** é uma aplicação web que permite:

* 📊 Registrar horas trabalhadas
* 📅 Visualizar registros por data
* 🧭 Navegar entre diferentes áreas do sistema
* 🔐 Simular autenticação de usuário

O projeto foi desenvolvido utilizando tecnologias modernas de frontend, com foco em performance e organização de código.

---

## 🚀 Tecnologias utilizadas

* React
* TypeScript
* Vite
* Tailwind CSS

---

## 📦 Pré-requisitos

Antes de começar, você precisa ter instalado:

* Node.js (versão 18 ou superior)
* npm (gerenciador de pacotes)

---

## ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/furia-hours-tracker.git
```

Acesse a pasta do projeto:

```bash
cd furia-hours-tracker
```

Instale as dependências:

```bash
npm install --legacy-peer-deps
```

---

## ▶️ Como executar o projeto

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Após isso, acesse no navegador:

```
http://localhost:8080
```

ou a porta exibida no terminal.
